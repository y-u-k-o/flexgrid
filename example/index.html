Simple Syntax

col-xs-6

        <div class="content"></div>

</div>

Responsive

Responsive modifiers enable specifying different column sizes, alignment and distribution at xs, sm, md viewport widths. Responsive Grids work by adding media classes to the Grid cells or containers. When those media values are met, the grids automatically adjust accordingly.

col-xs-12 col-sm-6 col-md-3
col-xs-6 col-sm-6 col-md-3
col-xs-6 col-sm-12 col-md-6
col-xs-12 col-sm-6 col-md-4
col-xs-12 col-sm-6 col-md-8
col-xs-6 col-sm-12 col-md-8
col-xs-6 col-sm-12 col-md-4
<div class="row">
    <div class="col-xs-12 col-sm-10 col-md-8">
        <div class="content">Responsive</div>
    </div>
</div>
